# Visual References

Drop screenshots and images here that represent the look and feel you want
your app to achieve. Claude Code will study these before making design decisions.

## What to include

- **Full-page screenshots** of apps you admire (Linear, Vercel, Stripe, Notion, etc.)
- **Component close-ups** showing card treatments, navigation, forms, or data tables you like
- **Typography samples** showing heading/body combinations you want to replicate
- **Color palette references** or mood boards
- **Your own app screenshots** showing the current state (so Claude can see what needs fixing)

## File naming convention

Name files descriptively so Claude understands context:

```
linear-dashboard-card-layout.png
stripe-pricing-page-spacing.png
vercel-sidebar-navigation.png
my-app-current-homepage.png
desired-card-style-reference.png
```

## Supported formats

PNG, JPG, WEBP — any standard image format that Claude can view.
