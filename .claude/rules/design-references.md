## Design References — ENFORCED

Two design analogs for all UI work in this app:

1. **Apple Notes** — Super clean, uncluttered navigation. Zero visual noise. If a screen feels cluttered, simplify.
2. **Monday.com** — Bold color usage for maximum impact, interesting transitions, polished list structure. Closest to the target look and feel.

### Critical Rule
Use RSNE's own brand colors (navy, brand-blue, brand-orange) with the same boldness and intentionality that Monday.com uses theirs. Do NOT adopt Monday.com's color palette.

### How to Apply
- Navigation feels cluttered? → Simplify like Apple Notes
- Colors feel flat/timid, lists feel generic? → Study Monday.com's color blocks, left accent bars, transitions — then apply with RSNE colors
- Cards should have status-colored left accent bars (4px border-l)
- Status badges at /15 opacity minimum (not /8 — invisible in sunlight)
- Touch targets 44px minimum (construction workers with gloves)
- All transitions duration-300, smooth spring curves
